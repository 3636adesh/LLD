# 🧭 5-Month LLD Roadmap (Java Focus)

### 🎯 Purpose
This roadmap is designed for a **Java backend developer (3+ years experience)** to master Low-Level Design (LLD), write scalable code, and crack LLD interview rounds.

🕒 Recommended time: **2–3 hours/day (parallel with DSA)**  
📌 Approach: **1 LLD problem per week + iterative improvement**

---

## 📅 Month 1 — LLD Foundations (OOP & Clean Design)

**🎯 Goal:** Build strong fundamentals in object-oriented design.

**📘 Topics**
- SOLID Principles
- OOP Concepts (Encapsulation, Inheritance, Polymorphism, Abstraction)
- Class Relationships (Association, Aggregation, Composition)
- Basic UML (Class Diagrams)

**🧩 Weekly Problems**
- Week 1: Parking Lot System 🚗
- Week 2: Library Management System 📚
- Week 3: Tic Tac Toe Game ❌⭕
- Week 4: Logger Framework 🪵

**🏁 Milestone**
> Design clean class structures and write maintainable code.

---

## 📅 Month 2 — Object Modeling & Data Flow

**🎯 Goal:** Improve system modeling and introduce flexibility.

**📘 Topics**
- Interfaces & Abstraction
- Strategy Pattern
- Basic Persistence Thinking
- Java Collections usage in design

**🧩 Weekly Problems**
- Week 5: LRU Cache
- Week 6: File System
- Week 7: Vending Machine 🥤
- Week 8: Splitwise System 💸

**🏁 Milestone**
> Model real-world systems with proper abstractions.

---

## 📅 Month 3 — Design Patterns & Backend Thinking

**🎯 Goal:** Apply design patterns in real systems.

**📘 Topics**

**🧩 Weekly Problems**
- Week 9: Notification System 🔔
- Week 10: Rate Limiter 🚦
- Week 11: Ride Booking System 🚕

> Use design patterns effectively in LLD problems.

**🧩 Weekly Problems**

- Week 16: Multiplayer Game Lobby 🎮
